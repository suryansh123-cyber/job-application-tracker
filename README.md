# Job Application Tracker (JobTrack Pro)

A professional personal job application tracking system built with the MERN stack. Designed for students and job seekers to manage their career journey in one place.

## 🚀 Features
- **Modern Dashboard**: Visual stats overview of application progress.
- **Application Management**: Full CRUD for job applications with status tracking.
- **Resume Management**: Integrated local file upload for managing resumes per application.
- **Secure Auth**: JWT-based authentication with protected routes.
- **Responsive UI**: Glassmorphic, student-friendly design using Tailwind CSS.

## 🛠️ Tech Stack
- **Frontend**: React, Tailwind CSS, Axios, Lucide Icons.
- **Backend**: Node.js, Express, MongoDB, Mongoose, Multer, JWT.

---

## 🚦 How to Run

### 1. Prerequisites
- Node.js installed.
- MongoDB installed and running locally on `mongodb://localhost:27017`.

### 2. Backend Setup
1. Navigate to the `server` directory:
   ```bash
   cd server
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. (Optional) Update `.env` if your MongoDB configuration is different.
4. Start the server:
   ```bash
   npm start
   ```
   *Server will run on http://localhost:5000*

### 3. Frontend Setup
1. Navigate to the project root:
   ```bash
   cd ..
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
   *Frontend will run on http://localhost:3000*
